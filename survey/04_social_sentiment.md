# ソーシャルメディア・トレンド分析 (2026 Q1)

## 概要
X (旧Twitter) や Reddit 上でのゲーマーの会話、バイラルツイートの傾向を分析しました。
2026年初頭現在、**「Friendslop (友人関係を泥沼化させるゲーム)」** や **「High Stakes Comedy (リスクが生むコメディ)」** がトレンドワードとして浮上しています。

## 1. X (Twitter) トレンド: "絶叫と爆笑"
ゲーマーの間でバイラル化しやすいクリップには共通点があります。
- **「完璧な作戦が、些細なミスで崩壊する」瞬間**:
    - *Lethal Company* のようなCo-opゲームで、誰かが足を滑らせて全員が巻き添えになる動画が依然として人気。
- **「予想外の裏切り」**:
    - 意図的なPvPではなく、Co-op中に発生する「事故的な裏切り（Friendly Fireや、脱出ポッドの定員オーバー）」が、"Clippable" (切り抜き動画にしやすい) コンテンツとして消費されています。

## 2. Reddit ディスカッション: "Co-op Poker" への渇望
r/boardgames や r/gaming のスレッドでは、協力型ポーカーに関する議論が散見されます。
- **"The Gang" (ボードゲーム)**: 「会話禁止でポーカー役を協力して作る」というボドゲが話題。
- **デジタルへの移植要望**: 「これをオンラインで、かつ *Lethal Company* のような没入感でやりたい」という潜在需要が存在します。

## 3. 2026年の注目ワード: "R.E.P.O." と "Peak"
- **R.E.P.O.**: 2025-2026年にかけて話題になった「最も面白いホラーゲーム」。ホラーとコメディの境界線が曖昧になっている現在のトレンドを象徴しています。
- **Peak**: 協力プレイにおけるコミュニケーションの不全を楽しむトレンド。

## 結論: "Death High Roller" への示唆
ソーシャルトレンドは完全に **「シリアスなギャンブル」ではなく「ギャンブルが生むカオスな状況」** を求めています。
- **方向性の微修正**:
    - ストイックにチップを稼ぐだけでなく、**「負けたプレイヤーを誰が見捨てるか」** といった政治的（Political）なメカニクスを入れると、配信者界隈で爆発する可能性が高いです。
    - 例: 「脱出ポットは3つしかないが、プレイヤーは4人いる。誰を置いていくかをポーカーで決める」
