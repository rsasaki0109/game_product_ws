# Bubble Drifter Design

## One-line
シャボン玉に包まれて空を流れ、割るタイミングを読んでリングを抜ける空中レース。

## Core Loop
1. ロビーで人数待ち
2. カウントダウン後に雲のスタート台へスポーン
3. Bubble で浮き、風レーンに乗ってリングへ向かう
4. Pop で落下して次のラインへ合わせる
5. 高いリングを多く抜けたプレイヤーが勝つ

## MVP Rules
- プレイヤー数: 1-8
- 1ラウンド: 80秒
- スコア: 抜けたリング数 + ゴール到達順
- 脱落: 奈落に落ちたらスタート近くへ戻す

## Controls
- `Q`: Bubble
- `E`: Pop
- 移動: 通常の `WASD`

## Fantasy
- シャボン玉の中は少し曇って見える
- 風レーンは色とパーティクルで進行方向が一目で分かる
- Bubble 中は足が少し浮いて見える

## MVP Scope
- 雲のスタート台
- 3本の風レーン
- 5つのリング
- 1つのゴール台
- Bubble / Pop のみ
- ラウンド HUD

## Not In MVP
- 他プレイヤーを弾く
- バブルの種類違い
- コスメ課金
- ステージ分岐

## Next Build Target
- Bubble 中だけ上向きの力と横ドリフトを与える
- Pop 後 0.5 秒は再展開不可
- リング通過をサーバーで集計
