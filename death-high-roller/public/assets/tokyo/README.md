## Tokyo City アセット置き場

`ThreeTokyoCity` は以下のファイル名を自動読込します。  
必要なら同名の glTF(.glb) を置いてください。

- `tokyo_tower.glb`
- `shibuya_crossing.glb`
- `tokyo_station.glb`
- `tokyo_tower_small.glb`

導入フロー（おすすめ）

- `public/assets/tokyo/` フォルダを作成
- `GLB` を上記ファイル名のどれかにリネーム
- 画面を `?mode=tokyo&asset=1` で開く

- 例: Sketchfab の検索  
  - `https://sketchfab.com/search?license=creative_commons&type=models&q=tokyo%20building`
- 例: Poly Pizza（CC0 寄せ）  
  - `https://poly.pizza/browse?text=tokyo`

上記はあくまで探索の起点です。  
ダウンロードした glb が重くないこと（1000～3000ポリ程度）を優先すると GitHub Pages でも軽く動きます。
