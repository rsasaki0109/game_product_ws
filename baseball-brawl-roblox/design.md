# Baseball Brawl (Roblox) — 実装版GDD

## コンセプト
- 2チーム制の乱闘サッカー/ベースボール風アクション
- ボールを殴り飛ばして相手ゴールへ叩き込む
- 乱闘、ノックバック、定期カオスイベントでクリップが生まれる設計

## ルール
- チーム: Red / Blue
- 勝利条件:
  - 先に `MAX_SCORE` 到達
  - または制限時間終了時に高得点
- ラウンド進行:
  - `Lobby -> Countdown -> Playing -> Result`

## 実装済みシステム

### 1. コアゲームプレイ
- 物理ボール (`BrawlBall`) の生成と中央スポーン
- ゴールセンサー接触による得点
- 得点後のキックオフリセット（全員再配置 + ボール再配置）
- ボール場外/落下時の自動リセット

### 2. バトルアクション
- `Swing`（LMB/F/R2）
  - 前方扇形にノックバック + スタン
  - ボール打撃（方向 + リフト + ブレ）
- `Dash`（E/Q/Shift/ButtonB）
  - 前方突進 + 近距離タックル
- ヒット時の画面シェイク通知

### 3. バズ導線
- 定期 `CHAOS PULSE`
  - 周囲プレイヤーとボールをまとめて吹き飛ばす
- `Bumper` ギミック
  - 接触プレイヤーを打ち上げ
- フィード通知
  - `GOAL`, `MONSTER SHOT`, `MULTI HIT`, `LAUNCHED`

### 4. 視認性/UI
- チーム別キャラハイライト
- HUD表示
  - 状態 / タイマー / スコア / ボール支配
  - スキルクールダウン
  - リアルタイムイベントフィード
- モバイルボタン（SWING / DASH）

## アリーナ構成
