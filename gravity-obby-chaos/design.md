# Gravity Obby Chaos — ゲーム設計書（GDD）

## 概要
**ジャンル**: マルチプレイ障害物レース × 重力カオス
**プラットフォーム**: Roblox
**プレイ人数**: 1〜20人（推奨 4〜12人）
**1プレイ時間**: 3〜5分（1ラウンド）

## コアコンセプト
> 「足元が天井になる障害物レース」

普通のObbyに見えるコースが、突然重力反転して天井が床になる。
壁を走り、天井を歩き、他プレイヤーの重力を狂わせながらゴールを目指す。

---

## ゲームフロー

### ラウンド構成
```
ロビー（待機）→ ステージ選択 → カウントダウン → レース → リザルト → ロビー
```

### ラウンド詳細
1. **ロビー（15秒〜）**: プレイヤーが集まるまで待機。ロビー内でも重力で遊べる
2. **カウントダウン（3秒）**: 全員スタート地点に配置
3. **レース（60〜120秒）**: Obbyを走る。重力イベントが発生
4. **リザルト（10秒）**: 順位発表、報酬配布

---

## 重力システム

### グローバル重力イベント
ステージ全体の重力が変わるイベント。

| イベント名 | 効果 | 頻度 |
|---|---|---|
| Gravity Flip | 重力が180°反転（天地逆転） | 15〜25秒間隔 |
| Gravity Shift | 重力が90°回転（壁が床に） | 20〜30秒間隔 |
| Zero-G | 無重力状態（3秒間） | 1ステージに1〜2回 |
| Gravity Storm | 2秒ごとにランダム方向へ切替 | ボスステージのみ |

### ローカル重力（アイテム）
プレイヤーが使える重力操作アイテム。

| アイテム | 効果 | 対象 |
|---|---|---|
| 重力爆弾 | 半径内の全員の重力をランダム方向に変える | 周囲のプレイヤー |
| 重力アンカー | 次の重力シフトを無効化（自分だけ） | 自分 |
| 反重力ブースト | 3秒間浮遊して高速移動 | 自分 |
| 重力リバース | 対象1人の重力を反転 | 指定プレイヤー |

### 重力ゾーン（マップギミック）
マップ上に固定配置される重力異常エリア。

- **反転ゾーン**: 入ると重力が反転
- **無重力トンネル**: 浮遊しながら通過する区間
- **重力渦**: 中心に引き寄せられる / 弾き飛ばされる
- **方向固定ゾーン**: グローバル重力イベントの影響を受けない安全地帯

## ステージ設計

### Stage 1: 「はじめての無重力」（チュートリアル）
- 難易度: ★☆☆☆☆
- 重力イベント: Gravity Flip × 1回のみ（中間地点で発生）
- アイテム: なし
- 目的: 重力反転の仕組みを体験させる
- 構成: 通常Obby → 反転 → 天井を歩いてゴール

### Stage 2: 「グラビティタワー」
- 難易度: ★★★☆☆
- 重力イベント: Gravity Shift（20秒間隔）
- アイテム: 重力爆弾、重力アンカー
- 構成: 垂直タワーを登る。重力シフトで壁が通路になったり、通路が壁になる
- 特徴: 4面ある壁それぞれにルートがある（どの重力方向でも進める設計）

### Stage 3: 「カオスアリーナ」（ボスステージ）
- 難易度: ★★★★★
- 重力イベント: Gravity Storm（2秒ごとにランダム）
- アイテム: 全種類
- 構成: 広いアリーナ型。固定ルートなし、重力を読んで最短距離を選ぶ
- 特徴: 生き残ること自体が難しい。足場が少なく、空中制御が重要

## プレイヤーの操作

| 操作 | PC | モバイル |
|---|---|---|
| 移動 | WASD | バーチャルスティック |
| ジャンプ | Space | ジャンプボタン |
| アイテム使用 | E / クリック | アイテムボタン |
| カメラ | マウス | スワイプ |

### 重力変化時の挙動
- 重力方向が変わると、キャラクターの「足元」が新しい重力方向に合わせて回転
- カメラも0.3秒かけてスムーズに追従（急激な回転は酔いの原因）
- 空中にいる時に重力が変わると、新しい方向に「落ちる」
- 着地時にパーティクルエフェクト + 画面揺れ（軽め）

```
┌──────────────────────────────────┐
│ [順位: 3/8]     [タイム: 01:23]  │
│                                  │
│           ゲーム画面              │
│                                  │
│ ⚠ GRAVITY SHIFT IN 3...         │
│                                  │
│ [アイテム: 重力爆弾 x1]  [コイン: 350] │
└──────────────────────────────────┘
```

### 重力変化の演出
1. 警告テキスト「⚠ GRAVITY SHIFT IN 3...2...1...」
2. 画面端にグリッチエフェクト
3. 重力切替の瞬間: 画面全体が一瞬歪む + SE
4. 新しい方向の矢印を0.5秒表示

## バイラル要素（バズるための仕掛け）
- **キルカム風リプレイ**: 重力変化で吹き飛ばされるシーンを自動録画
- **「今日の重力」**: 毎日変わる特殊ルール（例: 重力5倍、常時Zero-G）
- **クリップ共有**: ゴール時 or 死亡時にスクリーンショット自動生成
- **実績バッジ**: 「天井を100m歩いた」「無重力で30秒滞空した」等
